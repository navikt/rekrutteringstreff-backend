# Utviklerrutiner

Denne mappen inneholder rutiner som utviklere må følge ved vedlikehold og utvikling av Rekrutteringstreff.

> **Viktig:** Disse dokumentene er referert til i ROS-analysen og skal holdes oppdatert ved endringer i rutiner eller tiltak.

## Innhold

| Dokument                       | Beskrivelse                                 | ROS-referanser                                                                     |
| ------------------------------ | ------------------------------------------- | ---------------------------------------------------------------------------------- |
| [ki-rutiner.md](ki-rutiner.md) | Rutiner for vedlikehold av KI-sjekken (ROB) | 29337, 29330, 29263, 29025, 29023, 28415, 27868, 27867, 27854, 27853, 27852, 27544 |

## Formål

Utviklerrutiner dokumenterer:

- **Manuelle prosesser** som må følges ved spesifikke hendelser
- **Tiltak fra ROS-analysen** som ikke kan automatiseres
- **Beste praksis** for vedlikehold av systemet
- **Tilgangsstyring** og rollebruk for utviklere
- **Testing og benchmarking** av KI-sjekken

Rutinene er koblet til ROS-analysen via ROS-ID-referanser.
